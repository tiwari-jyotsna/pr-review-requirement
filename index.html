Overview

The AI PR Review Agent is an automated system that analyzes GitHub Pull Requests using Large Language Models (LLMs). The agent reviews code changes, identifies issues, and posts structured feedback directly on the Pull Request.

The system supports multiple LLM providers and runs locally as a Python-based automation tool.

Functional Requirements

  • Read GitHub Pull Requests using GitHub token authentication.
  • Fetch changed files and code diffs.
  • Send PR changes to LLM models for analysis.
  • Generate AI-based code review.
  • Provide severity classification.
  • Post inline comments on specific code lines.
  • Post a summary review comment on the PR.
  • Support multiple LLM providers.
  • Run locally using Python scripts on Windows machines.

Processing Workflow


1. User provides GitHub PR URL
2. Script authenticates using GitHub Token
3. Fetch PR details and file changes
4. Build LLM review prompt
5. Send diff content to selected LLM
6. Generate AI review feedback
7. Produce summary + inline comments
8. Post review back to GitHub PR

Core System Components

Component Description
GitHub Adapter Handles authentication and API interaction with GitHub
PR Data Collector Retrieves changed files, diffs, and metadata
Context Builder Prepares structured prompts for LLM analysis
LLM Router Selects the configured LLM provider
Review Engine Generates summary review and inline comments
Review Publisher Posts review results back to GitHub PR

Supported LLM Providers

  • OpenAI (GPT models)
  • Google Gemini
  • Anthropic Claude
  • AWS Bedrock models

The architecture uses a provider abstraction layer to enable easy switching between models.
